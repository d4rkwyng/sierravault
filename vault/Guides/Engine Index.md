# Engine Index

<small style="color: gray">Last updated: May 8, 2026</small>

Complete listing of Sierra games organized by engine, with technical specifications and compatibility information. Use ScummVM for most adventure games, DOSBox for simulations and sports titles.

### AGI (Adventure Game Interpreter) — 1984–1989

Sierra's first standardized engine, created for King's Quest I on the IBM PCjr.

| Spec | Details |
|------|---------|
| Resolution | 160×200 (displayed 320×200) |
| Colors | 16 (EGA/CGA) |
| Interface | Text parser |
| Audio | PC Speaker, Tandy 3-voice, Apple IIgs |
| RAM | 256KB required |
